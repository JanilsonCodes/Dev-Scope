## 📌 About The Project

**Dev Scope** is a frontend project built to study and improve my skills in working with APIs, especially the GitHub API.

The application allows users to search and discover developers, view their profiles, repositories, and highlight users with the most followers.

This project has no financial purpose — it was built purely for learning and improving my development skills.

---

## 🎯 Goal

- Learn how to consume real APIs  
- Improve JavaScript skills  
- Work with asynchronous requests  
- Understand GitHub API structure  
- Build a functional real-world interface  

---

## ⚡ Features

- GitHub API integration  
- Developer search system  
- Profile display (followers, repos, etc.)  
- Trending users (most followed)  
- Dynamic content rendering  
- Responsive design
